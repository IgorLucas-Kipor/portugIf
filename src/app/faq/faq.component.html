Perguntas Frequentes

Para quem essa plataforma é destinada?

A plataforma PortugIF foi pensada e desenhada para aqueles que estão iniciando estudos de programação e possuem dificuldades com conceitos ou desejam reforçar seus aprendizados, com informações oferecidas em forma simplificada e linguagem coloquial justamente para facilitar o entendimento por parte desse público.

Por que o código é fornecido em Portugol?

Existem muitas diferenças entre linguagens de programação, das funcionalidades disponíveis, para a forma como são utilizadas, para até mesmo a declaração de variáveis. Uma vez que a plataforma não deseja se focar em uma plataforma ou outra, mas sim passar uma visão mais geral para estudantes, optou-se por disponibilizar códigos em Portugol, tanto para facilitar a compreensão do código pelos usuários quanto pelo fato de ser uma pseudo-linguagem utilizada para fins letivos.

Há planos de se expandir a plataforma?

Trabalhos futuros nunca estão fora de cogitação e sugestões são bem-vindas e podem ser livremente enviadas por meio do e-mail disponibilizado no rodapé dessa página, mas não há planos imediatos de expansão da plataforma.
